【项目说明】
这是 Scm.Net 的客户端应用，支持 Windows、macOS 和 Linux。

【使用方法】
1、下载最新版本的客户端应用；
2、解压到任意目录；
3、根据操作系统，双击运行对应的可执行文件：

- Windows：scm.net-\*.exe
- macOS：scm.net-mac-\*
- Linux：scm.net-linux-\*

【代码开发】
1、安装 Node.js；
2、安装 Neutralinojs CLI：
npm install -g @neutralinojs/neu
3、进入项目目录：
cd app/
4、运行项目：
neu run
5、编译项目：
neu build
